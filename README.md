# Playwright Automation Daniel Portfolio

Modern end-to-end testing with **Playwright + TypeScript**

### Tech Stack
- Playwright Test (@playwright/test)
- TypeScript
- GitHub Actions CI/CD
- HTML Report + Video on failure

### How to run
```bash
npm install
npx playwright test          # headless (CI)
npx playwright test --headed  # visible browser
npx playwright test --ui      # amazing GUI mode
